# Iran-War-Energy-Prices-Analysis

🛢️ Iran War Energy Market Impact Analysis Dashboard
Interactive Streamlit Dashboard Project
Analyzing the impact of geopolitical conflict on oil, gas prices, and shipping activity.
🎯 Project Objective
Analyze how the Iran War affected Brent, WTI, and Dubai crude oil prices, U.S. gasoline prices, and Strait of Hormuz shipping activity using an interactive Streamlit dashboard.
📊 Dashboard Features
• KPI Cards
• Oil Price Timeline
• Gas Price Analysis
• War Phase Analysis
• Correlation Heatmap
• Key Events Timeline
• Interactive Filters
🛠️ Technologies Used
Python, Pandas, Plotly, Streamlit, NumPy, Data Visualization
📈 Key Insights
• Oil prices increased during conflict periods.
• Gasoline prices rose across regions.
• Shipping activity changed after the war began.
• Strong relationships can be observed between energy markets and logistics activity.
